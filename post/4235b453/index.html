摄影学习笔记（二）———— Pr快捷键

发表于 2019-02-22 分类于 三分技术 浏览： 阅读时长 ≈ 1 分钟
PR相关的快捷键

视频来源B站影视飓风

  1. 快速转场

    1. Shift + D 音频和视频的转场
    2. Ctrl + D 单独添加视频转场
    3. Ctrl + Shift + D 单独添加音频转场
    4. 在效果面板中可以设置默认的转场类型

  2. 波纹剪辑

    1. Q 和 W
    2. Shift + Q 和 Shift + W 延长素材

  3. 匹配侦

    1. F 在素材监视器上找到对应的帧
    2. Shift + R 反向匹配帧

  4. 缩放时间轴

    1. +/-缩放时间轴
    2. \ 快速缩放
    3. Ctrl + \ 隐藏顶部项目信息

  5. 移动剪辑 1.Alt + 方向键 2.Alt + Shift + 方向键 快速移动

  6. 抽取与删除

    1. I/O 设定区域
    2. ; 删除
    3. ' 抽取

  7. 轨道缩放

    1. Ctrl + 加号减号
    2. Shift + 加号减号 一次性缩放所有轨道

  8. 迅速切换窗口 Shift+123

  9. 其他快捷键

    视频来源B站南道

    1. V 选择箭头
    2. A 往前选中
    3. Shift + A 往后选中
    4. Ctrl + K 编辑
    5. Ctrl + Alt + V 超级黏贴

  10. 制作电影宽荧幕遮幅

    调整图层 效果 搜索裁剪 上11.8 下11.8 添加关键帧 动态效果
