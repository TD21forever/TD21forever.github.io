  1. 快速转场
    1. Shift + D 音频和视频的转场
    2. Ctrl + D 单独添加视频转场
    3. Ctrl + Shift + D 单独添加音频转场
    4. 在效果面板中可以设置默认的转场类型

< !–more–>

  1. 波纹剪辑

    1. Q 和 W
    2. Shift + Q 和 Shift + W 延长素材

  2. 匹配侦

    1. F 在素材监视器上找到对应的帧
    2. Shift + R 反向匹配帧

  3. 缩放时间轴

    1. +/-缩放时间轴
    2. \ 快速缩放
    3. Ctrl + \ 隐藏顶部项目信息

  4. 移动剪辑 1.Alt + 方向键 2.Alt + Shift + 方向键 快速移动

  5. 抽取与删除

    1. I/O 设定区域
    2. ; 删除
    3. ' 抽取

  6. 轨道缩放

    1. Ctrl + 加号减号
    2. Shift + 加号减号 一次性缩放所有轨道

  7. 迅速切换窗口 Shift+123
